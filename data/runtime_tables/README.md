# Runtime Relative CSVs

这些 CSV 是当前仓库生成的相对路径版主表，默认假设命令从仓库根目录执行。

- `flow_train_rel.csv`：Flow 训练主表（相对路径版）
- `flow_val_rel.csv`：Flow 验证主表（相对路径版）
- `flow_infer_test536_rel.csv`：Flow clean test(536) 推理输入表
- `posecred_ipg_train_docked_rel.csv`：PoseCred-IPG train docked 主表（相对路径版）
- `posecred_ipg_val_docked_rel.csv`：PoseCred-IPG val docked 主表（相对路径版）
