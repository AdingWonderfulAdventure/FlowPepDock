# `scripts/ops/`

本目录存放 **工程运维辅助脚本**。

## 包含内容

- `note_guard.py`
  - 检查代码改动后 notes 是否同步更新
- `run_with_bark.py`
  - 包装长任务执行，并在满足阈值时发送 Bark 通知
- `extract_markdown_math_to_word.py`
  - 提取指定 `Markdown` 文档中的公式、数学符号与下划线变量名，并同步导出汇总 `docx`

## 使用场景

- 开发时保证文档 / 台账不失同步
- 跑长任务时做简单通知与工程守护

## 备注

- 这类脚本和模型本身无直接算法关系
- 主要用于保持仓库协作规范和长任务可追踪
